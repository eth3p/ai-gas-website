ChatGPT-4使用完全指南：从新手到高级技巧

April 10, 2026 · 1 min read · Categories: reviews tutorials

ChatGPT-4是目前最强大的AI对话模型之一。本文将教你如何充分发挥它的潜力。

📖 基础使用

1. 注册与设置

  • 访问 chat.openai.com 注册账号
  • 选择 ChatGPT Plus 订阅（每月$20）
  • 设置自定义指令，让 ChatGPT 记住你的偏好

2. 界面功能介绍

  • 对话历史：左侧面板保存所有对话
  • 模型切换：GPT-4、GPT-4 Turbo、GPT-3.5
  • 插件市场：200+官方插件可选
  • 文件上传：支持图片、PDF、Word、Excel等

  • 创建专属 GPTs：编程助手、写作教练、数据分析师

3. API 集成

import openai

client = openai.OpenAI(api_key="your-api-key")
response = client.chat.completions.create(
    model="gpt-4-turbo",
    messages=[
        {"role": "system", "content": "你是一位专业的文案写手"},
        {"role": "user", "content": "写一篇关于AI工具指南的引言"}
    ]
)
